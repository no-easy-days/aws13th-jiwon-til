# 📝 배운 내용 (Today I Learned)

## 변수 
- 값을 저장하는 이름
- = 연산자를 사용해 값 할당
- 파이썬은 변수 선언 시 자료형을 따로 지정하지 않음 (자동으로 자료형이 결정)

## 파이썬의 자료형

#### int (정수형)

- 소수점이 없는 정수 값
- 사칙연산이 가능함

---

#### float (실수형)

- 소수점을 포함한 숫자
- 계산 결과가 실수가 될 수 있음

---

#### str (문자열)

- 문자나 문자열을 저장하는 자료형
- 작은따옴표(' ') 또는 큰따옴표(" ") 사용

---

#### bool (논리형)

- 참(True) 또는 거짓(False)을 나타냄
- 조건문에서 주로 사용됨

---

#### list (리스트)

- 여러 값을 순서대로 저장
- 중복 허용, 값 수정 가능

---

#### tuple (튜플)

- 여러 값을 순서대로 저장
- 값 수정이 불가능

---

#### dict (딕셔너리)

- key와 value 쌍으로 데이터 저장
- key를 통해 value에 접근

---

#### set (집합)

- 중복된 값을 허용하지 않음
- 순서 개념이 없음

---
