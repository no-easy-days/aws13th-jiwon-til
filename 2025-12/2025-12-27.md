## 조건문과 반복문 (if, for, while)

### 조건문 (if)
- 조건이 참일 때만 코드를 실행한다
- 조건에 따라 서로 다른 동작을 수행할 수 있다

```python
score = 85

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C")
````

- if는 첫 조건을 검사한다
- elif는 앞 조건이 거짓일 때만 검사된다
- else는 모든 조건이 거짓일 때 실행된다

---

### 반복문 (for)

- 정해진 횟수나 여러 개의 값들을 순서대로 반복할 때 사용한다

```python
for i in range(5):
    print(i)
```

```python
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)
```

- range()는 숫자 범위를 만들어준다
- 리스트, 문자열 등 반복 가능한 객체에서 자주 사용된다

---

### 반복문 (while)

- 조건이 참인 동안 계속 반복된다

```python
count = 0

while count < 3:
    print(count)
    count += 1
```

- 반복 횟수가 정해져 있지 않을 때 유용하다
- 조건이 계속 참이면 무한 루프가 될 수 있으므로 주의해야 한다

---

## 함수 정의와 호출

### 함수란?

- 반복되는 코드를 하나로 묶어 재사용하기 위한 기능이다
- 코드의 가독성과 유지보수를 높여준다

---

### 함수 정의

```python
def add(a, b):
    return a + b
```

* def 키워드로 함수를 정의한다
* 매개변수는 함수에 전달되는 값이다
* return은 함수의 결과값을 반환한다

---

### 함수 호출

```python
result = add(3, 5)
print(result)
```

- 함수 이름 뒤에 괄호를 사용해 호출한다
- 전달한 값은 인자라고 한다

---

### 함수 사용 예시

```python
# 함수 정의
def add(a, b):
    return a + b

# 함수 호출
sum_value = add(10, 20)
print(sum_value)
```
