## 기본 입출력 (input, print)

### 출력 (print)
- print() 함수는 값을 화면에 출력할 때 사용한다

```python
print("Hello, Python")
print(10 + 5)
````

- 여러 값을 쉼표로 구분해 출력할 수 있다

```python
name = "wonzz"
age = 20

print(name, age)
```

---

### 입력 (input)

- input() 함수는 사용자로부터 값을 입력받는다
- 입력받은 값은 항상 문자열(str) 형태이다

```python
name = input("이름을 입력하세요: ")
print("안녕하세요,", name)
```

---

### 입력값 형 변환

- 숫자로 사용하려면 형 변환이 필요하다

```python
age = int(input("나이를 입력하세요: "))
print(age + 1)
```

- int(), float() 등을 사용해 자료형을 변환할 수 있다

## 클래스(Class)

### 클래스란?

- 클래스는 변수와 함수를 하나로 묶은 설계도이다
- 객체를 만들기 위한 틀 역할을 한다

---

### 클래스 정의

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
```

* class 키워드로 클래스를 정의한다
* **init** 메서드는 객체가 생성될 때 자동으로 실행된다
* self는 생성된 객체 자신을 의미한다

---

### 클래스 메서드 정의

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def introduce(self):
        print("이름:", self.name)
        print("나이:", self.age)
```

- 클래스 안에서 정의된 함수는 메서드라고 한다
- self를 통해 객체의 변수에 접근한다
