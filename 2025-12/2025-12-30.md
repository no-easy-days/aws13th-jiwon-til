## 클래스(class)
- 관련된 데이터와 기능을 하나로 묶어 코드를 정리하는 도구
- 변수가 늘어날수록 관리가 불가능해집니다

```python
class Student:
    def __init__(self, name, student_id, score):
        self.name = name
        self.student_id = student_id
        self.score = score
    
    def is_passed(self):
        return self.score >= 60
    
    def get_grade(self):
        if self.score >= 90:
            return "A"
        elif self.score >= 80:
            return "B"
        elif self.score >= 70:
            return "C"
        else:
            return "F"

# 학생 30명도 깔끔하게 관리
cheolsu = Student("김철수", "2024001", 85)
younghee = Student("이영희", "2024002", 92)

print(cheolsu.get_grade())  # B
print(younghee.is_passed())  # True
````


- if는 첫 조건을 검사한다
- elif는 앞 조건이 거짓일 때만 검사된다
- else는 모든 조건이 거짓일 때 실행된다

## 람다(lambda)

- 한 번 쓰고 버릴 간단한 함수를 이름 없이 바로 만들 수 있습니다.
- 한 줄 표현식만 가능합니다.
- 복잡한 로직이 필요하면 **일반 함수**를 사용해야합니다.

```python
lambda 매개변수: 반환할_표현식
```
### 일반 함수와 비교
```python
# 일반 함수
def square(x):
    return x ** 2

# 람다 함수
square = lambda x: x ** 2

# 둘 다 같은 결과!
print(square(5))  # 25
```

### lambda 함수의 활용처
- max() / min() - 최대/최소 기준 지정하는 경우
- map() - 모든 요소에 함수 적용하는 경우
- filter() - 조건에 맞는 요소만 추출하는 경우
