## CSV
- CSV (Comma-Seperated-Values): 쉼표로 구분된 데이터 형식(엑셀과 호환)
- 표 형태, 액셀에서 바로 열림
- 엑셀 데이터를 Python으로 분석 시 사용
## CSV 함수 정리

| **작업** | **함수** | **설명** |
| --- | --- | --- |
| 읽기 (리스트) | `csv.reader(f)` | 각 행을 리스트로 반환 |
| 읽기 (딕셔너리) | `csv.DictReader(f)` | 각 행을 딕셔너리로 반환 |
| 쓰기 (리스트) | `csv.writer(f)` | 리스트를 CSV로 저장 |
| 쓰기 (딕셔너리) | `csv.DictWriter(f, fieldnames)` | 딕셔너리를 CSV로 저장 |

## JSON
- JSON (JavaScript Object Notation): 키-값 쌍으로 구성된 데이터 형식(웹 API 표준)
- 계층 구조, 프로그래밍에 친화적
- 카카오/네이버 API에서 데이터를 받거나, AWS, Docker 설정 파일을 다룰 시 사용

### JSON 파일 읽기
```python
import json

# 파일에서 JSON 읽기
with open('user.json', 'r', encoding='utf-8') as f:
    data = json.load(f)  # 파일 → 딕셔너리

print(data['name'])   # 홍길동
print(data['skills']) # ['Python', 'AWS', 'Docker']
```
### JSON 문자열 파싱(API) 응답 처리
```python
import json

# API에서 받은 JSON 문자열이라고 가정
json_string = '{"name": "홍길동", "age": 25}'

# 문자열 → 딕셔너리
data = json.loads(json_string)  # load"s" = load string
print(data['name'])  # 홍길동
```
### JSON 파일 쓰기
```python
import json

user = {
    'name': '홍길동',
    'age': 25,
    'city': '서울',
    'skills': ['Python', 'AWS', 'Docker']
}

### 딕셔너리 → JSON 파일
with open('user.json', 'w', encoding='utf-8') as f:
    json.dump(user, f, ensure_ascii=False, indent=2)
```

| **작업** | **함수** | **설명** |
| --- | --- | --- |
| 파일 → 딕셔너리 | `json.load(f)` | JSON 파일 읽기 |
| 문자열 → 딕셔너리 | `json.loads(s)` | JSON 문자열 파싱 |
| 딕셔너리 → 파일 | `json.dump(obj, f)` | JSON 파일 쓰기 |
| 딕셔너리 → 문자열 | `json.dumps(obj)` | JSON 문자열 생성 |
